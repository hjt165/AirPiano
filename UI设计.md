# Air Piano UI 设计说明书

## 1. 界面布局

### 1.1 主界面

```
+------------------------------------------+
|           Air Piano (标题栏)              |
+------------------------------------------+
|                                          |
|                                          |
|        摄像头实时预览 (全屏)              |
|                                          |
|     [手部关键点叠加层 - 黄色点+青色线]    |
|                                          |
|                                          |
+------------------------------------------+
|  |  Do  |  Re  |  Mi  |  Fa  |  Sol  |  |  ← 琴键区域 (18% 屏幕高度)
|  | 白色 | 白色 | 白色 | 白色 | 白色  |  |
+------------------------------------------+
|  ▲ 下拉展开设置面板                       |
+------------------------------------------+
```

### 1.2 设置面板（底部 Bottom Sheet）

从底部上拉展开，包含：
- 推理时间显示
- 检测阈值调节（-/+ 按钮）
- 追踪阈值调节（-/+ 按钮）
- 存在阈值调节（-/+ 按钮）
- 推理后端选择（CPU/GPU 下拉框）

### 1.3 导航栏

底部导航栏包含两个 Tab：
- **Camera**：实时摄像头模式（主功能）
- **Gallery**：图片/视频识别模式

## 2. 配色方案

### 2.1 琴键配色

| 状态 | 颜色 | 色值 |
|------|------|------|
| 空闲 | 白色 | #FFFFFF |
| 空闲边框 | 深灰 | #444444 |
| Do (拇指) 按下 | 暖红 | #FF6B6B |
| Re (食指) 按下 | 浅橙 | #FFA07A |
| Mi (中指) 按下 | 金色 | #FFD700 |
| Fa (无名指) 按下 | 浅绿 | #98FB98 |
| Sol (小指) 按下 | 天蓝 | #87CEEB |

按下状态琴键文字为白色，空闲状态为深灰色。

### 2.2 手部关键点配色

| 元素 | 颜色 |
|------|------|
| 关键点 | 黄色 (#FFFF00) |
| 连接线 | 青色 (mp_color_primary) |
| 线宽 | 8px |
| 点样式 | 填充圆点 |

### 2.3 背景

无固体背景色，直接显示摄像头实时画面。

| 元素 | 字号 | 样式 |

| 琴键标签 | 36sp | 加粗，居中 |

## 4. 交互说明

### 4.1 手部检测
- 手进入画面 → 21 个关键点以黄色圆点显示，青色连线连接
- 手离开画面 → 关键点和连线消失，琴键保持空闲状态

### 4.2 敲击交互
- 手指向下敲击 → 对应琴键填充为独特颜色 + 播放音符
- 手指保持姿态 → 琴键持续亮起（HOLD 状态）
- 手指抬起 → 琴键恢复白色 + 状态转为 RELEASE

### 4.3 快速连续敲击
- 同一手指快速连续敲击 → 每次敲击重新播放音符（seekTo(0)）
- 多根手指同时敲击 → 多个琴键同时亮起，多个音符同时播放

### 4.4 设置调节
- 下拉底部面板 → 展开设置选项
- 点击 -/+ 按钮 → 调整对应阈值（步长 0.1，范围 0.1-0.9）
- 切换 Delegate → 重新初始化推理引擎

## 5. 响应式设计

- 琴键宽度：屏幕宽度 / 5，等宽分布
- 琴键高度：屏幕高度的 18%
- 摄像头预览：4:3 比例，FILL_START 模式
- 手部关键点坐标：归一化坐标 × 图像尺寸 × 缩放因子

## 6. 无障碍设计

- 琴键标签使用高对比度文字颜色
- 按下状态使用明显颜色变化
- 设置面板按钮有 contentDescription 属性
